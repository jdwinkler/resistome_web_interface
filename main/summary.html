The visualization above is a circular representation of the E. coli pangenome (containing B, W, K). Linkages in the center represent gene co-occurence (green) or differential expression co-occurence in the same genotype, whereas each successive shell represents coverage of SNPs or AA replacements, amplifications, deletions, inversions, and indels described within Resistome records.

Data Integration and Analysis Methods:

The Resistome incorporates a wide range of complementary data sources and tools:

  • Biocyc integration
  • COBRA modeling via Cobrapy
  • SNAP2 mutation effect prediction
  • INPS mutation effect prediction
  • Circos visualization
  • Interaction data from EcoliNet V1 and RegulonDB
  • Gene deletion-metabolite relationships
While this website provides limited search capabilities, the distribution below contains more extensive functionality to make use of these additional features.

Data Downloads

The complete Resistome software distribution, along with instructions on how to build your own copy of the database, can be obtained here. Two summary datasets are available: a compilation of each individual mutation in the database with associated phenotypes and annotated mutation data downloadable here and a count of how many times a given gene is mutated and total set of phenotypes those gene mutations are associated with here. If there is a particular dataset that would be useful for your research not provided here, please do not hesitate to contact us for customized data extraction.
