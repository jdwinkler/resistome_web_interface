Data Integration and Analysis Methods

The Resistome incorporates a wide range of complementary data sources and tools:

  • Biocyc integration
  • COBRA modeling via CobraPy
  • Phenotype prediction via Tensorflow
  • SNAP2 mutation effect prediction
  • INPS mutation effect prediction
  • Circos visualization
  • Interaction data from EcoliNet V1 and RegulonDB
  • Gene deletion-metabolite relationships
  • The Adaptive Laboratory Evolution database
While this website provides limited search capabilities, the distribution below contains more extensive functionality to make use of these additional features. Phenotype prediction approaches may be able to make use of the SQL database dump linked below, although please see the note below concerning the Resistome data cleaning process.

Usage Notes

The Resistome is comprised of data that is curated directly from scientific publications, usually from the supplementary tables defining mutant genotypes and their associated phenotypes. You should expect that there will be errors in this dataset stemming from ambiguities in the original publications or the curation process. We did not attempt to correct/normalize the provided relative mutation genome positions into their absolute positions for strains other than E. coli MG!655, REL606, and W. You can likely obtain gene position calls from NCBI for the relevant strains to do this correction, and we may do so in the future. Alternatively, we may repeat the resequencing and variant calling process to normalize the variant calls per strain.

Data Downloads

The complete Resistome software distribution, along with instructions on how to build your own copy of the database, can be obtained here. A PostgresSQL dump is also available here. Two summary datasets are available: a compilation of each individual mutation in the database with associated phenotypes and annotated mutation data downloadable here and a count of how many times a given gene is mutated and total set of phenotypes those gene mutations are associated with here. If there is a particular dataset or data format that would be useful for your research not provided here, please do not hesitate to contact us for customized data extraction.
